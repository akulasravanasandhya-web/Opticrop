# Opticrop
Smart Agricultural Production Optimization Engine (OptiCrop) is a Machine Learning-based crop recommendation system that predicts the most suitable crop using soil nutrients (N, P, K), temperature, humidity, pH, and rainfall. Built with Python, Flask, Scikit-learn, Pandas, and NumPy.
# Smart Agricultural Production Optimization Engine (OptiCrop)

## Project Overview
OptiCrop is a Machine Learning based crop recommendation system that predicts the most suitable crop based on soil nutrients and environmental conditions.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

## Features
- Crop Recommendation
- Soil Analysis
- User-Friendly Interface

## Input Parameters
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## Output
Recommended crop based on the given inputs.
